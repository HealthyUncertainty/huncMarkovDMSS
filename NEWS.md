# huncMarkovDMSS News

## huncMarkovDMSS 0.1.0

* Initial release.
* 4-state population proportion Markov model: 5 treatment arms.
* Frozen grade distributions after treatment response phase.
* Perioperative mortality for surgical arms (Septoplasty, Ablatherm).
* Surgical disutilities with lifetime pacemaker probabilities.
* 4-week cycles, 80-year horizon.
* PSA across 23 parameters (Beta, Gamma, Utility distributions).
* Multi-comparator ICER table using dampack CE frontier.
* Interactive Shiny app with CE plane, CEAC, and trace for all 5 arms.
* Developed using HEPackageR skill for Claude AI.
